🚀 Installation Guide

1

Enable Developer Mode

Navigate to Settings → System → Advanced system settings → Developer options and enable both "Installer" and "Debugger".

2

Download & Prepare

Clone the repository and create a deployment package with the provided zip command.

3

Deploy to Roku

Access your Roku's developer interface via web browser and upload the KickRokuApp.zip file.
