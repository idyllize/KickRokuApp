Get Started

Install KickRokuApp on your Roku device in just a few simple steps.

1

Enable Developer Mode

Access your Roku's developer settings by entering the sequence: Home x3, Up x2, Right, Left, Right, Left, Right.

2

Download the App

Clone the KickRokuApp repository from GitHub and package the app using the provided build tools.

git clone https://github.com/idyllize/KickRokuApp.git
3

Install & Launch

Upload the packaged app to your Roku device via the developer portal and launch it to start streaming.
